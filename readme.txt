BioGraph — Análisis de Interacciones Proteicas para el Descubrimiento de Fármacos

Este proyecto modela redes de interacción proteína-proteína (PPIN) como grafos no dirigidos
e implementa algoritmos (BFS, Dijkstra, centralidad de grado) para identificar proteínas
esenciales (hubs) que son dianas terapéuticas en el desarrollo de fármacos.
